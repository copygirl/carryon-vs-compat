## Carry On + Valkyrien Skies Compat [moved to git.gay](https://git.gay/copygirl/carryon-vs-compat)

This mod patches [Carry On] to allow you to pick up blocks from [Valkyrien Skies] ships.

[Carry On]: https://github.com/Tschipp/CarryOn
[Valkyrien Skies]: https://github.com/ValkyrienSkies/Valkyrien-Skies-2

### Development

The following files need to be present in `libs/` to build:
- [`carryon-forge-1.20.1-2.1.2.7.jar`](https://cdn.modrinth.com/data/joEfVgkn/versions/edGQD16r/carryon-forge-1.20.1-2.1.2.7.jar)
- [`valkyrienskies-120-2.3.0-beta.5.jar`](https://cdn.modrinth.com/data/V5ujR2yw/versions/JmkvaGb4/valkyrienskies-120-2.3.0-beta.5.jar)
